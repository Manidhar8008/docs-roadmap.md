# Project Roadmap

## Objective
Establish a scalable automation pipeline for job applications, smart filtering, resume tailoring, and activity logging.

## Current Architecture
- Data ingestion module
- JD parsing + keyword extraction
- Resume tailoring engine
- Auto-apply workflow
- Google Sheets logger
- Deployment module (TBD)

## Upcoming Enhancements
- Add notifications
- Integrate ATS score predictor
- Add daily activity scheduler
- Improve fail-safe retry logic
- Add analytics dashboard

## Tech Stack
Python, Selenium/Playwright, FastAPI, Google Sheets API, GitHub Actions

## Deployment Plan
1. Local prototype
2. Modularization
3. Containerization
4. Cloud orchestration (Render/AWS/Lambda)
